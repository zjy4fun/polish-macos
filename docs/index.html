选中文本，确认后
一键润色

Polish 把“复制粘贴 + 多轮改写”压缩成一个动作： `⌥⌘P` 唤起确认窗口，编辑后点击“开始润色”，立即得到简化版本、优化版本和 Commit Message。

macOS 13+ 默认 Codex CLI 缓存命中，秒回结果 菜单栏恢复最近结果
去 Releases 下载 查看文档
1. 选中文本 在任意 app 里选中内容后按 `⌥⌘P`。
2. 确认编辑 在确认窗口里先改好原文，再点“开始润色”。
3. 输出即用 切换查看简化版 / 优化版 / Commit Message，一键复制。

核心能力

确认后再执行

不是盲目立刻调用模型。先给你一个编辑确认步骤，减少误触与误润色。

本地优先默认

默认使用 Codex CLI，也可切换 OpenAI API 或 Claude Code CLI。

命中缓存

相同文本 + Provider 配置自动命中缓存，避免重复等待与重复请求。

快速开始

cd macos
swift build
swift run PolishMac
